SEtoolbox

Streamline Your SummarizedExperiment Workflows in R, WOW!

🔬 Why SEtoolbox?

A specialized R package for manipulating and analyzing SummarizedExperiment objects. Designed for genomics researchers who need:

  • ⚡️ Simplified data management for high-throughput assays
  • 🔍 Built-in QC metrics and visualization
  • 📊 Seamless integration with Bioconductor ecosystem
  • 📈 Production-ready differential analysis pipelines

📦 Installation

Stable Release (Bioconductor)

if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
BiocManager::install("SEtoolbox")

Development Version (GitHub)

devtools::install_github("shaoxunyuan/SEtoolbox")

🚀 Quick Start

Basic Workflow

library(SEtoolbox)
library(SummarizedExperiment)

# Load example dataset
data(se)

# Quality control
se <- run_qc(se, plot=TRUE)

# Differential analysis
de_results <- find_markers(se, 
                          group_col = "condition",
                          method = "DESeq2")

# Visualize results
volcano_plot(de_results)

Explore more in the Getting Started vignette.

✨ Key Features

📥 Data I/O

  • Import from GEO/SRA
  • Convert to/from SingleCellExperiment
  • Multi-assay support

🔧 Core Utilities

  • Subset by row/column metadata
  • Batch effect correction
  • Row/column statistics

📈 Analysis Modules

  • Differential expression (DESeq2/edgeR/limma)
  • Pathway enrichment analysis
  • Dimension reduction (PCA/t-SNE/UMAP)

📊 Visualization

  • Interactive QC dashboards
  • MA/Volcano plots
  • Heatmap clustering
