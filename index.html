Installation

To install SEtoolbox, you can use Bioconductor. Run the following command in your R console:

if (!requireNamespace("BiocManager", quietly = TRUE))  
    install.packages("BiocManager")  
BiocManager::install("SEtoolbox")

Usage

After installation, load the package and start using it:

library(SEtoolbox)  
example_SE <- loadSE()  
# Perform your analysis here

For more in-depth examples and functions, refer to the vignette included in the package:

vignette("SEtoolbox")

Features

  • Load and manage SummarizedExperiment objects effortlessly.
  • Provide comprehensive analysis functions tailored for genomic data.
  • Integrate with existing Bioconductor packages for enhanced functionality.
  • Vignettes and documentation for easy reference and guidance.
